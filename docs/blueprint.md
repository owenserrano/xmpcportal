# **App Name**: XMPC Technologies B2B Portal

## Core Features:

- Corporate Home Page: A statically generated home page (SSG) with a professional hero banner, service highlights, and a clear call to action.
- Detailed Service Pages: In-depth descriptions of XMPC's services (IT consulting, cybersecurity, managed services) to demonstrate expertise and attract corporate clients.
- B2B Product Showcase: A product gallery displaying B2B-focused equipment (servers, network infrastructure, security appliances), sourced from a local JSON file or the SQLite database.
- Contact Form with API Route: A functional contact form handled by a Next.js API route to process and store inquiries in the SQLite database using Prisma.
- Client Registration & Login: Backend logic (API Routes) for client registration and secure login to manage accounts and track purchase history.
- Shopping Cart Functionality: API routes that handles adding products to cart, product quantities, persisting items to the SQLite database.

## Style Guidelines:

- Primary color: XMPC Red (#D90429) will be used sparingly for key CTAs, links, and accents.
- Background color: Light gray (#F8F9FA) as a secondary background for sections to create visual separation on the main white background.
- Accent color: Darker shade of gray (#212529) to complement the primary red, and provide a high level of readability on text.
- Body and headline font: 'Inter', a sans-serif font, will be used throughout the website.
- Use a professional icon set from React Icons or Heroicons, particularly for the service sections, to visually represent different IT solutions.
- Maintain a clean and corporate layout, prioritizing whitespace and intuitive navigation, drawing inspiration from https://tux-solutions.com/.
- Subtle transition effects can be applied on elements while hover action is detected.
